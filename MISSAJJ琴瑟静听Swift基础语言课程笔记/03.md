# 循环
 
##循环的介绍

- 在开发中经常会需要循环
- 常见的循环有:for/while/do while.
- 这里我们只介绍for/while,因为for/while最常见
## OC风格的 `for`循环

```swift
// 传统写法
for var i = 0; i < 10; i++ {
    print(i)
}
```

## Swift风格的 `for`循环

```swift
// 遍历 0 ~ <10
for i in 0..<10 {
    print(i)
}

print("---")

// 遍历 0 ~ 10
for i in 0...10 {
    print(i)
}
```

* 阶段性小结
    * `Swift` 中使用 `in` 关键字标示循环的范围
    * `0..<10` 表示从0到9
    * `0...10` 表示从0到10
    * 注意之间不能出现空格

## 特殊写法
- 如果在for循环中不需要用到下标i

```swift
for _ in 0...10 {
    print("hello")
}
```

* 阶段性小结
    * 如果不关心循环本身的索引，可以使用 `_` 忽略
    * 这一技巧在之前的分支演练中出现过


## while和do while循环

- while循环
  - while的判断句必须有正确的真假,没有非0即真
  - while后面的()可以省略

```swift
var a = 0
while a < 10 {
    a++
}
```

- do while循环
  - 使用repeat关键字来代替了do

```swift
let b = 0
repeat {
    print(b)
    b++
} while b < 20
```
